# Rayzz Unban Website ini menyediakan layanan pemulihan akun media sosial dengan keamanan anti-cyber attack dan anti-DDoS. ## Fitur - **Pemulihan Akun Media Sosial**: Instagram, Facebook, Twitter, TikTok. - **Keamanan Anti-Cyber Attack**: Sanitasi input, reCAPTCHA, dan prepared statements. - **Anti-DDoS Attack**: Batasi jumlah request per IP. ## Cara Menggunakan 1. Clone repositori ini. 2. Buka file `index.html` di browser. 3. Isi formulir pemulihan akun. ## Author **@Rayzz1232**
